# 자연꽃꿀 (Natural Flower Honey) — 온라인 상점

국내산 천연벌꿀 100% 판매 쇼핑몰. 제철 꿀(아카시아·감로 등)을 스틱·통 단위로 판매합니다.

## 구성
- **고객 화면**: 홈 / 상품목록 / 상품상세 / 장바구니 / 주문 / 주문조회 / FAQ
- **관리자 화면** (`/admin`): 대시보드 / 상품 관리(등록·수정·삭제) / 주문 관리

## 현재 단계
정적 프론트엔드 (HTML/CSS/JS). 데이터는 브라우저 localStorage에 저장됩니다.
`assets/js/store.js`의 데이터 함수만 백엔드 API(fetch) 호출로 교체하면 실제 온라인 상점이 됩니다.

## 로드맵
1. ✅ 프론트엔드 (현재) — Vercel 배포
2. ⬜ 백엔드 API (Express) — Railway 배포
3. ⬜ 데이터베이스 (PostgreSQL) — Neon
4. ⬜ 결제 연동 (PG) + 도메인
